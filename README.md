# Review Reply Orchestrator

This repository contains a background automation service that manages periodic review checking and reply posting. It features a Google My Business review monitoring service.

## Files
- `automation_service.py`: Background automation orchestration.
- `monitor.py`: Google My Business review monitoring service.
- `.env.example`: Example environment configuration file.
- `README.md`: This documentation file.

## Usage
1. Clone the repository.
2. Copy `.env.example` to `.env` and configure it.
3. Run the automation service using the appropriate command.
